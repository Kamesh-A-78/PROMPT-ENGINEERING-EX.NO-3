# Ex.No.3-Scenario-Based Report Development Utilizing Diverse Prompting Techniques for the the following Prompt Engineering types with examples -Straightforward Prompts - Tabular Format Prompting - Missing Word Prompting - Preceding Question Prompting.
# REG NO.: 212223060110
# Aim: To write the prompts for these following prompt types andevaluate that using any one method 1. Straightforward Prompts 2. Tabular Format Prompting 3.Preceding Question Prompting 4. Missing Word Prompting
# Use Case: Education System
Artificial Intelligence can help students learn concepts easily, generate
summaries, create tables, answer questions, and improve interactive
learning experiences.
# Procedure:
# 1. Straightforward Prompts
Straightforward prompts directly ask the AI for a specific response
without additional context.
Example Prompts
• “Define Artificial Intelligence in one sentence.”
• “List three advantages of online learning.”
# Generated Output
• Artificial Intelligence is the simulation of human intelligence by
machines.
• Advantages of online learning:
1. Flexible learning schedule
2. Easy access to resources
3. Cost-effective education
# 2. Tabular Format Prompting
Tabular prompting requests information in table format for better
comparison and readability.
Example Prompt
“Provide a table comparing Online Learning and Classroom
Learning.”
# Generated Output
 Feature Online Learning Classroom Learning
Flexibility High Limited
Interaction Virtual Face-to-face
Cost Lower Higher
Accessibility Anywhere Specific location
# 3. Preceding Question Prompting
This prompting method asks a leading question before requesting the
main explanation.
Example Prompt
“Why is digital education important? Explain how online platforms
help students learn effectively.”
# Generated Output
Digital education is important because it provides flexible and
accessible learning opportunities. Online platforms help students by
offering recorded lectures, interactive quizzes, and easy access to
study materials from anywhere.
# 4. Missing Word Prompting
Missing word prompting asks the AI to fill in the blanks with suitable
answers.
Example Prompts
• “The process of learning through the internet is called ______.”
• “Artificial Intelligence is commonly abbreviated as ______.”
# Generated Output
• Online Learning
• AI
# RESULT: 
The various types of Prompts are executed successfully
with generated the report.
